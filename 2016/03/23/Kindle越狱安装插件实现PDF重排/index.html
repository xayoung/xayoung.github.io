Kindle越狱安装插件实现PDF重排

发表于 2016-03-23   |   分类于 Read   |     |   阅读次数

前言

Kindle作为阅读的神器，集齐舒服的水墨屏、精美排版的电子书、持久舒适的阅读体验等诸多优点，但是其对PDF尤其是扫描版支持堪忧，阅读体验相对亚马逊官方的电子书简直无法忍受，之前有提到过使用GAE部署KindleEar实现自动推送新闻博客文章，接下来将再进一步，对Kindle进行越狱，实现自动重排PDF。

对比图

越狱

首先下载好文件
jb.zip
JailBreak-1.14.N-FW-5.x-hotfix.zip

第一步

  • 解压缩，得到 jb 文件夹，把里面的 jb 文件拷贝到 Kindle根目录下；
  • 从电脑弹出 Kindle，然后点击 Kindle右上角的菜单键，打开“体验版网页浏览器”；
  • 点击地址栏调出键盘，在地址栏中输入网址 http://kindlefere.com/jb，回车；
  • 在打开的页面中，点击底部的【I Agree】按钮，切换到另外一个页面；
  • 先点击页面下方的链接 Stage1，看到出错页面后点击“返回按钮”返回；
  • 再点击页面下方的链接 Stage2，当看到顶部状态栏出现“Run ;fc-cache in the search bar.”时，点击“搜索按钮”，并输入“;fc-cache”（不带引号，只输入标识为红色的字符，别漏掉分号“；”），回车；
  • 当你看到“Jailbreak succeeded!”的提示信息，即表示操作成功。

第二步

  • 解压缩下载到的 ZIP 压缩包 JailBreak-1.14.N-FW-5.x-hotfix.zip，得到一个名为 Update_jailbreak_bridge_1.14.N_install.bin 的文件；
  • 将 bin 文件拷贝到 Kindle 磁盘根目录；
  • 依次在 Kindle 中点击“菜单 —> 设置 —> 菜单 —> 更新您的 Kindle”，等待重启；
  • 重启完毕后 Kindle 应该处于越狱状态了。

插件

KUAL
插件下载（KUAL-v2.6.zip）： 官方链接

安装步骤：

  • 将 KUAL-KDK-2.0.azw2 拷贝到 Kindle 的 Documents 文件夹中；
  • 打开 Kindle，即可看到一个名 Kindle LAUNCHER图标。

MobileRead Package Installer (MRPI) —— 插件安装器

插件下载（kual-mrinstaller-1.6.N.zip）：官方下载

安装步骤：

  • 将 extensions 和 mrpackages 拷贝到 Kindle 根目录下。

Koreader —— PDF 重排软件

需要安装 KUAL 及其插件 MRinstaller，再安装 kpvbooklet，最后安装 Koreader。

插件下载（kpvbooklet ）：官方下载
插件下载（Koreader ）：官方下载

安装步骤：

首先确保安装了 KUAL 及其插件 MRPI；

  • 将kpvbooklet文件夹内的 update_kpvbooklet_0.6.2_install.bin 拷贝到 Kindle 里 mrpackages 文件夹中；
  • 点击 Kindle 中的 kindle LAUNCHER，依次点击 Helper -> Install MR Packages；
    耐心等待 kpvbooklet 安装，直到安装完成后 Kindle 重启完毕；
  • 将Koreader文件夹 extensions 中的内容拷贝到 Kindle 根目录下的 extensions 文件夹中；
  • 将文件夹内的 koreader 文件夹拷贝到 kindle 根目录下，即安装完毕。
